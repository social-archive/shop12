## 🚀 [실습 프로젝트] 여행 상품 판매 서비스 만들기 (프론트엔드)

---

### 📝 프로젝트 개요

React와 Context API를 활용하여 여행 상품을 판매하는 **웹 서비스**를 구현하였습니다.

**외부 API**를 사용하여 데이터를 가져오고, **자동화 테스트 및 배포까지 직접 경험**해봅니다.

### 🎯 주요 학습 영역 (Learning Objectives)

* **Context API를 활용한 상태 관리**
* **RESTful API 통신 및 데이터 처리**
* **테스트 자동화**
* **서비스 배포 및 통합**

### 🏗️ 산출물 (Deliverables)

* **[설계]** 서비스 아키텍처 설계 문서
* **[개발]** API 기능 명세서
* **[배포]** AWS 배포 결과물 링크 (URL 제공)
